# Δαπάνη Ειδικών Αστυφυλάκων — Μηνιαία μεταφορά PDF → Excel

Εφαρμογή γραμμής εντολών (Python) που κάθε μήνα διαβάζει **οπτικά** τη σαρωμένη
επιστολή της Αστυνομίας Κύπρου προς τον ΟΚΥπΥ μέσω του Claude API (vision — όχι
tesseract/OCR, ώστε τα χειρόγραφα ✓ δίπλα στα ποσά να αγνοούνται σωστά),
επαληθεύει τα ποσά έναντι των συνόλων-ελέγχου της επιστολής και προσθέτει το
μπλοκ του μήνα στο φύλλο **«Ανάλυση»** του workbook
`ΧΡΕΩΣΕΙΣ ΕΙΔΙΚΟΙ ΑΣΤΥΦΥΛΑΚΕΣ <έτος>.xlsx`, χωρίς να πειράξει το ΣΥΝΟΠΤΙΚΟ
ή άλλες φόρμουλες.

## Δομή του workbook που περιμένει

- **«Ανάλυση»**: ενιαίος πίνακας όλων των μηνών — μία γραμμή ανά άτομο ανά
  μήνα, κενή γραμμή-διαχωριστικό ανάμεσα στους μήνες. Στήλες:
  `A ΜΗΝΑΣ | B Α/Α | C ΑΚΑ | D ΑΔΤ | E Ε/ΑΣΤ | F Ονοματεπώνυμο | G Ημ. Τοποθέτησης |
  H ΝΟΣΗΛΕΥΤΗΡΙΟ | I Βασικός | J Τιμάριθμος | K–N (παλιές αυξήσεις) | O Αύξηση |
  P Βάρδια | Q Κυριακή/Αργία | R =SUM(I:Q) | S =R×24,25% | T =R+S | U =T×10% | V =T+U`
- **«ΣΥΝΟΠΤΙΚΟ»**: SUMIFS σε ολόκληρες στήλες του «Ανάλυση» (ανά νοσηλευτήριο
  και μήνα) — οι νέες γραμμές πιάνονται αυτόματα, δεν το αγγίζουμε ποτέ.

## Εγκατάσταση

```bash
python -m venv .venv && source .venv/bin/activate   # προαιρετικά
pip install -r requirements.txt
export ANTHROPIC_API_KEY=sk-ant-...                 # ποτέ hardcoded στον κώδικα
```

Απαιτείται Python 3.11+.

## Χρήση

```bash
python -m daspani "Επιστολή_Απρίλιος_2026.pdf" "ΧΡΕΩΣΕΙΣ ΕΙΔΙΚΟΙ ΑΣΤΥΦΥΛΑΚΕΣ 2026.xlsx"
```

| Επιλογή | Περιγραφή |
|---|---|
| `--minas ΑΠΡΙΛΙΟΣ` | Επιβολή μήνα μισθών. Αν λείπει, διαβάζεται από την επιστολή. |
| `--istoriko ΠΕΡΣΙΝΟ.xlsx` | Περσινό workbook για συμπλήρωση ΑΔΤ/νοσηλευτηρίου στους πρώτους μήνες της χρονιάς. |
| `--sto-idio` | Εγγραφή στο ίδιο αρχείο (μετά από επιβεβαίωση) αντί για νέο αντίγραφο `<όνομα>_<ΜΗΝΑΣ>.xlsx`. |
| `--model ...` | Μοντέλο Claude (προεπιλογή `claude-sonnet-4-6`, ή μεταβλητή `DASPANI_MODEL`). |
| `--dpi 230` | Ανάλυση απόδοσης σελίδων σε εικόνα. |
| `--yes` | Χωρίς ερωτήσεις επιβεβαίωσης (για scripts). |

### Νέα χρονιά

```bash
python -m daspani.new_year "ΧΡΕΩΣΕΙΣ ΕΙΔΙΚΟΙ ΑΣΤΥΦΥΛΑΚΕΣ 2025.xlsx" "ΧΡΕΩΣΕΙΣ ΕΙΔΙΚΟΙ ΑΣΤΥΦΥΛΑΚΕΣ 2026.xlsx" --timarithmos "12,67%"
```

Κρατά ΣΥΝΟΠΤΙΚΟ/κεφαλίδες, αδειάζει το «Ανάλυση», αφαιρεί τις περσινές
χειροκίνητες προσαρμογές ±0,01 και τους περσινούς αριθμούς καταχώρησης.

## Τι κάνει βήμα-βήμα

1. **Rasterize** τις σελίδες με PyMuPDF σε ~230 dpi PNG. Οι σελίδες πίνακα
   (2 και μετά) περιστρέφονται +90°· αν είναι πολύ ψηλές, κόβονται σε πάνω/κάτω
   μισό με ~10% επικάλυψη και οι διπλές γραμμές αφαιρούνται (ΑΚΑ + ποσά).
2. **Σελίδα 1 → Claude API**: τα 8 σύνολα-ελέγχου (Βασικοί, Τιμαριθμικά,
   Αύξηση 1,5%, Βάρδια, Κυριακή/Αργία, Εισφορά, Διοικητικά 10%, Γενικό Σύνολο)
   και οι μήνες μισθών/επιδομάτων.
3. **Σελίδες πίνακα → Claude API**: όλες οι γραμμές σε JSON, μαζί με τις
   αναλογίες («ΑΠΟ/ΜΕΧΡΙ ...»), τις μη αριθμημένες διορθωτικές γραμμές
   (π.χ. «ΜΙΣΘΟΣ 3/2026» με αρνητικά ποσά) και τα «ΑΝΑΔΡΟΜΙΚΑ ΕΠΙΔΟΜΑΤΑ».
4. **Έλεγχος συνόλων** ανά στήλη έναντι της επιστολής (ανοχή ±0,05 €) **και**
   των παραγώγων της (Εισφορά = Σύνολο×24,25%, Διοικητικά = ×10%, Γενικό
   Σύνολο). Αν κάτι δεν κλείνει, μία αυτόματη επανάληψη ανάγνωσης σε 300 dpi.
5. **Έλεγχοι συνέπειας** ανά γραμμή (μόνο πλήρεις μήνες): Αύξηση ≈ Βασικός×1,5%,
   Τιμάριθμος ≈ (Βασικός+Αύξηση)×ποσοστό της επιστολής (12,67% κ.λπ.).
   Οι αποκλίνουσες γραμμές απλώς σημειώνονται για χειροκίνητο έλεγχο.
6. Το αποτέλεσμα του ελέγχου **εμφανίζεται πρώτα** — η εγγραφή γίνεται μόνο
   μετά από επιβεβαίωση.
7. **Εγγραφή**: νέο μπλοκ μήνα στο τέλος του «Ανάλυση» (με κενή γραμμή
   διαχωρισμού), φόρμουλες R–V ανά γραμμή, μορφοποίηση αντιγραμμένη από την
   τελευταία υπάρχουσα γραμμή. Το ΑΔΤ (στήλη D — δεν υπάρχει στο PDF) και το
   νοσηλευτήριο (στήλη H) συμπληρώνονται από τις παρατηρήσεις του PDF ή από την
   πιο πρόσφατη προηγούμενη γραμμή του ίδιου ΑΚΑ (ή από το `--istoriko`).
   Ό,τι δεν βρεθεί, αναφέρεται για χειροκίνητη συμπλήρωση.
8. **Idempotency**: αν ο μήνας υπάρχει ήδη, το μπλοκ του αντικαθίσταται μετά
   από επιβεβαίωση — δεν διπλογράφεται ποτέ.
9. **Τελική αναφορά**: γραμμές, OK/διαφορά ανά στήλη, ΑΔΤ/νοσηλευτήρια που
   λείπουν, γραμμές για χειροκίνητο έλεγχο.

## Δομή κώδικα

```
daspani/
  __main__.py       CLI και κύρια ροή
  new_year.py       Δημιουργία workbook νέας χρονιάς από το περσινό
  pdf_images.py     PDF → PNG (PyMuPDF), περιστροφή, τεμαχισμός με επικάλυψη
  claude_extract.py Κλήσεις Claude API (vision), parsing JSON, συγχώνευση
  models.py         Δομές δεδομένων, parsing ποσών/μηνών/ημερομηνιών, νοσηλευτήρια
  verify.py         Έλεγχοι συνόλων, παραγώγων και συνέπειας
  excel_write.py    Εγγραφή μπλοκ μήνα στο «Ανάλυση» (openpyxl)
  report.py         Εμφάνιση ελέγχου και τελικής αναφοράς
tests/              Unit tests (pytest)
```

## Σημειώσεις

- Κόστος ανά μήνα: 3 εικόνες ανά τρέξιμο ≈ λίγα cents.
- Τα API calls γίνονται με temperature 0 και αυτόματο retry με backoff.
- Το ΣΥΝΟΠΤΙΚΟ ενημερώνεται μόνο του (SUMIFS σε ολόκληρες στήλες) με το
  άνοιγμα του αρχείου στο Excel.
- Οι χειροκίνητες προσαρμογές ±0,01 στο ΣΥΝΟΠΤΙΚΟ (συμφωνία στρογγυλοποιήσεων
  με τις καταχωρήσεις) παραμένουν δουλειά του χρήστη, όπως και οι αριθμοί
  «ΚΑΤΑΧΩΡΗΣΗ ΔΑΠΑΝΗΣ».
