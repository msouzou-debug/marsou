On iPhone / iPad Your phone
  1. Open this page in Safari (not another browser).
  2. Tap Open the app above, or the Share button (the square with an up-arrow).
  3. Scroll down and tap Add to Home Screen, then Add.
  4. The brass Clean Shot icon appears — tap it to run full-screen, even offline.
On Android Your phone
  1. Open this page in Chrome.
  2. Tap Install app when it pops up — or the ⋮ menu → Install app / Add to Home screen.
  3. Confirm Install.
  4. The Clean Shot icon appears in your app drawer / home screen.
